# GetContractCount
Gets the count of contracts
<hr>

### Request

> POST https://testneofura.ngd.network:444

#### Body Parameters

none


#### Example
```
curl --location --request POST 'https://testneofura.ngd.network:444' \
--header 'Content-Type: application/json' \
--data-raw '{
  "jsonrpc": "2.0",
  "method": "GetContractCount",
  "params": {},
  "id": 1
}'
```
### Response
```json5
{
    "id": 1,
        "result": {
        "total counts": 518
    },
    "error": null
}
```
