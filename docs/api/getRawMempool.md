# GetRawMempool
Gets the transactions in the memory pool
<hr>

### Request

> POST https://testneofura.ngd.network:444

#### Body Parameters

none


#### Example
```
curl --location --request POST 'https://testneofura.ngd.network:444' \
--header 'Content-Type: application/json' \
--data-raw '{  
  "jsonrpc": "2.0",
  "method": "GetRawMemPool",
  "params": {},
  "id": 1
}'
```
### Response
```json5
{
    "id": 1,
        "result": [],
        "error": null
}
```
